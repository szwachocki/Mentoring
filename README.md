# O projekcie

Projekt testowy do nauki i ćwiczeń z TypeScript dla testerów.

## Jak korzystać?

Możesz uruchamiać różne moduły używając skryptów z pliku `package.json`

## Jak przygotowano ten projekt?

Inicjalizacja projektu node.js:

```
npm init -y
```

Instalacja typescript oraz tsx do uruchamiania skryptów:

```
npm install typescript tsx
```

Aby utworzyć plik tsconfig.json:

```
npx tsc --init
```

## Przestarzałe
Dla Node 18 możesz użyć ts-node, jednak nie będzie działać z Node 20
